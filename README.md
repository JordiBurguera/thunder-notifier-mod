# ⚡ Thunder Alerter

A Minecraft mod that uses **Discord Webhooks** to instantly notify you when a thunderstorm begins. Perfect for AFK sessions while waiting for charged creepers or rare mob heads.

---

## 📥 1. Download the Mod
Download the latest version of the mod from Modrinth:

## 🔧 2. Discord Webhook Setup
To receive notifications, you need to create a bridge between Minecraft and Discord:
1. If you don't have an account, sign up at [Discord.com](https://discord.com).
2. In your Discord server, go to **Channel Settings** (gear icon ⚙️ next to the channel name).
3. Go to **Integrations** > **Webhooks** > **New Webhook**.
4. Click **Copy Webhook URL**. (Keep this link handy).

---

## 📂 3. Installation & Configuration

Follow the steps for your operating system to install the mod and link your Discord account:

### 🍎 On macOS
1. Close Minecraft if it is running.
2. In the **Finder** top menu bar, click on **Go**.
3. While the menu is open, hold down the **Option (⌥)** key. The **Library** folder will appear; click on it.
4. Navigate to: `Application Support` > `minecraft`.
5. Open the `mods` folder and paste the downloaded `.jar` file inside.
6. **Open Minecraft and then close it again** to allow the mod to generate its config files.
7. Go back to the `minecraft` folder, enter the **`config`** folder, and locate the file named `discord_webhook.txt`.
8. Open the file, delete any existing text, and **paste your Discord Webhook link**. Save and exit.

### 🪟 On Windows
1. Close Minecraft if it is running.
2. Press **Win + R** on your keyboard, type `%appdata%`, and hit Enter.
3. Open the `.minecraft` folder.
4. Open the `mods` folder and paste the downloaded `.jar` file inside.
5. **Open Minecraft and then close it again**.
6. Go back to the `.minecraft` folder, enter the **`config`** folder, and locate the file named `discord_webhook.txt`.
7. Open the file with Notepad, replace the text with your **Discord Webhook link**, and save the file.

---

## ✅ You're All Set!
Restart your game. From now on, whenever a thunderstorm starts, you will receive an instant notification on your Discord server.
